After fiddling with CSS, downloading every image on the internet, and getting cheat sheets with Copilot, I think this website is started to look like the way I want it to.

I will now go on about how things got here.

Before gemdation.com

www.gemdation.com was being used to show a Google Sites page

I don't like Google Sites for a few tiny reasons, mainly the heavy resource editor and the Report Site button. I don't think a report button looks good for any site.

I wanted to have a website I made that could be accessible

To go and do this, I made a tiny website called The Q that had websites for older browsers, aka, Nintendo browsers. I made it with the help of W3Schools and the amazing notepad.exe

I learned about Visual Studio Code

A wonderful tool that I used to code The Q and this website, I use it alongside GitHub Desktop. (no i do not trust the git support)

After gemdation.com

The website first started with a simple one page that explained that it was used for my sites to get rid of Google Sites being on the domain, it had simple index but got the big home page update in November.

The new update set the space theme of the website and was the "1.0" of it in a way.
